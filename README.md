# portfolio-website

Personal portfolio site for Anita Yan — work case studies, play projects, and about.

## Stack

- React + Vite
- React Router

## Local development

```bash
npm install
npm run dev
```

## Deploy (GitHub Pages)

The site is built automatically on every push to `main`.

**One-time Pages setup** in [repo Settings → Pages](https://github.com/anita3y/portfolio/settings/pages):

1. **Source:** Deploy from a branch
2. **Branch:** `gh-pages` → `/ (root)`
3. Save

Live URL: **https://anita3y.github.io/portfolio/**
